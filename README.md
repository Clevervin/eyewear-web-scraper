# eyewear-web-scraper
Python web scraping project that extracts eyewear frame and lens product data from e-commerce websites using BeautifulSoup and Selenium, transforming unstructured web data into structured datasets for analysis.

The scraper collects only publicly available product catalog information such as brand, product name, price, and product link. No personal, medical, or user account information is collected. The scraper accesses only publicly visible pages without authentication.
